# ⚽  GigioSBC

Un semplice gestore di giocatori per FUT / EA Sports FC.

## Funzionalità
- Aggiungi giocatori con nome, nazionalità, club, campionato, ruolo e rating.
- Visualizza la lista in tabella.
- (Prossimamente) Filtro e risoluzione di sfide con requisiti specifici.

## Uso
Apri `index.html` nel browser oppure pubblica con GitHub Pages.
